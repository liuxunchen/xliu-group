山东科技大学学位论文 LaTeX 模板 SDUST_Thesis

刘训臣

六月 20, 2026

简介

撰写学位论文时，使用 LaTeX 可以获得优雅的排版和高效的文献管理。然而，许多同学在寻找符合学校格式要求的模板时常常感到困扰。为此，我基于上海交通大学的 SJTUThesis 模板，修改了符合 山东科技大学研究生学位论文撰写规定 的 LaTeX 模板 —— SDUST_Thesis。

本模板已在学位论文写作中实际使用并通过测试，现分享给有需要的同学。

模板特点

  • ✅ 符合山东科技大学研究生学位论文撰写规范（字体、字号、行距、页边距、章节格式）
  • ✅ 支持 硕士、博士、学士 学位类别（通过文档类选项切换）
  • ✅ 使用 XeLaTeX 编译，支持 Unicode 和 OpenType 字体
  • ✅ 内置封面、版权声明、摘要、目录、章节、参考文献、附录、致谢、作者简历、学位论文数据集等完整结构
  • ✅ 参考文献使用 biblatex + biber，国标 GB/T 7714-2015 样式
  • ✅ 支持图表、算法、代码块、子图、双标题等常见排版需求
  • ✅ 提供 Emacs AUCTeX 本地变量支持，一键编译

下载与使用

下载方式

直接下载压缩包 下载 SDUSTThesis 模板 (.zip)

快速开始

  • 解压模板压缩包。
  • 编辑 main.tex，修改 \sdustsetup 中的个人信息（标题、作者、导师、专业等）。
  • 将 contents/ 下的各章示例内容替换为您自己的论文内容。
  • 在终端中执行以下命令编译：
xelatex main.tex
biber main
xelatex main.tex
xelatex main.tex

如果您使用 Emacs + AUCTeX，打开 main.tex 后按 C-c C-c 选择 XeLaTeX，再按 Biber，再两次 XeLaTeX 即可。

文件结构

SDUST_Thesis/
├── main.tex                 # 主文件
├── sdustthesis.cls          # 模板类文件
├── sdustlogo.png            # 校徽图片（封面用）
├── refs.bib                 # 参考文献数据库
├── contents/                # 各章节内容
│   ├── abstract.tex         # 摘要（中英文）
│   ├── variablelist.tex     # 变量注释表
│   ├── intro.tex            # 第1章 绪论
│   ├── chapter2.tex         # 第2章
│   ├── math_and_citations.tex
│   ├── floats.tex           # 图表示例
│   ├── summary.tex          # 总结与展望
│   ├── app_maxwell_equations.tex
│   ├── app_flow_chart.tex
│   ├── resume.tex           # 作者简历
│   └── acknowledgements.tex # 致谢
├── figures/                 # 独立绘图源文件
│   └── flow_chart.tex
├── assets/                  # 数据文件（如 CSV）
│   └── energy-distrib.csv
└── scans/                   # 扫描件（如已签名的版权声明）
    └── sample-copyright-a.pdf
