利用JANAF表估算氢氧火焰温度

高寅斌

八月 15, 2025

最近在着手准备有关JANAF表使用的相关论文，发现了一个有趣的问题，氢氧焰的燃烧温度应该怎么计算？这似乎并不难回答，按照热力学第一定律，反应前后的总焓保证不变即可，似乎很容易写出下面的热平衡：

$$\Delta_f H(H_2) + \frac{1}{2}\Delta_f H(O_2) = \Delta_f H(H_2O) + \Delta H(H_2O)$$

因为方程左边均为单质，这样看来水生成焓在数值上应该等于负的反应焓，查询JANAF表可以得知$\Delta_f H(H_2O)$等于-241.826kJ·mol⁻¹，我们可以查询多少温度下的焓变等于这个值的负值，结果是5200K，看起来这个问题就这么解决了，

但很明显这个答案与现实不符，一当量比的氢氧焰不可能有这么高的温度，在与老师讨论后发现了问题所在，

xliu: 这个道理没有解答本质问题。算一下，这个组分下，全部转化为H2O，是否 Gibbs 自由能达到了最低？【cantera 给组分就能算】焓是否守恒？

下面文章应该分为三层阐述：

一、只考虑一个反应，看不同温度下，是否Gibbs 自由能达到了最小？

HP 限制下，不同温度，根据组分守恒（K 和 组分的一次关系，也就是自由能最小）得到组分，看是否H守恒。 这就是扫描方法得到自由能最低点温度。

二、把温度也作为变量，和组分一起变。在HP 限制下，将温度作为变量进行迭代求解。cantera 中 温度也作为变量和元素势一起优化迭代。本文下面也是直接变化温度，手动迭代温度。

三、然后考虑更多的组分 O OH ，通过组分变换达到自由能最小。在组分的元素守恒条件下，变为一元二次方程。

考虑温度一般是从 以下两个入手： dU = T dS - p dV + \Sigma \mu dN dH = T dS + V dp + \Sigma \mu dN

通过约束 UV,或者 HP, 使得 dN 调整，使得熵最大，得到 T. Cantera 中6种 约束： TP, SP， SV ，TV 分别对应 4种自由能，再加上 HP 和 UV。 引用Gordon 报告。

水在高温下的分解没有被考虑，那么考虑分解结果如何呢？

下面来完整的算一下，所用到的数据均来自JANAF表：

由于$H_2O$在高温下会发生分解，则分别写出主要的分解方程式

反应1：

$$H_2 + \frac{1}{2}O_2 \to a1H_2O + b1H + c1OH$$

由元素守恒可得

$$\begin{cases}2 = 2a1 + b1 + c1 \\1 = a1 + c1 \end{cases} \Rightarrow \begin{cases}b1 = 1 - a1 \\c1 = 1 - a1 \end{cases}$$

反应1的分解为：

$$H_2O \to H + OH$$

该分解反应平衡常数

$$K = \frac{bc}{a} = \frac{(1 - a1)^2}{a1}$$

反应2：

$$H_2 + \frac{1}{2}O_2 \to a2H_2O + b2H_2 + c2O_2$$

由元素守恒可得

$$\begin{cases}2 = 2a2 + 2b2 \\1 = a2 + 2c2 \end{cases} \Rightarrow \begin{cases}b2 = 1 - a2 \\c2 = \frac{1 - a2}{2} \end{cases}$$

反应2的分解为

$$H_2O \to H_2 + \frac{1}{2}O_2$$

该分解反应平衡常数

$$K = \frac{b2c2^{\frac{1}{2}}}{a2} = \frac{(1 - a2)(\frac{1 - a2}{2})^{\frac{1}{2}}}{a2}$$

设产物温度为3100K

对反应1：查询JANAF表得：$\Delta_f H(H_2O) = -71298$ ，$\Delta_f H(H) = 39877$ ，$\Delta_f H(OH) = -5552$ ，则 $\Delta G_1 = 105623$

则

$$K = e^{-\frac{\Delta G}{RT}} = 0.0166$$

，由 $K = \frac{(1 - a1)^2}{a1}$ 求得 $a1 = 0.879$

即有0.121 mol的水发生了分解，产物为H和OH

对反应2：查询JANAF表得 $\Delta_f H(H_2O) = -71298$ ，$\Delta_f H(H_2) = \Delta_f H(O_2) = 0$ ，则 $\Delta G_2 = 71298$

则

$$K = e^{-\frac{\Delta G}{RT}} = 0.0629$$

，求得 $a2 = 0.825$

即有0.175 mol的水分解，产物为H₂和O₂

由热力学第1定律，反应前后总焓不变

写出热平衡

$$0 = [(1 - 0.121 - 0.175)(\Delta_f H(H_2O) + \Delta H(H_2O))] + [b1(\Delta_f H(H) + \Delta H(H))] + [c1(\Delta_f H(OH) + \Delta H(OH))] + [b2\Delta H(H_2)] + [c2\Delta H(O_2)]$$

求得 $0 > -2688.0705$

设产物温度3200K

同理，求得对反应1：$\Delta G_1 = 92313$ ，则

$$K = e^{-\frac{\Delta G}{RT}} = 0.0311$$

由 $K = \frac{(1 - a1)^2}{a1}$ 求得 $a1 = 0.8385$ ，即有0.1615 mol H₂O发生了分解

对反应2：$\Delta G_2 = 65430$ ，则 $K = 0.0855$

由 $K = \frac{(1 - a2)(\frac{1 - a2}{2})^{\frac{1}{2}}}{a2}$ 求得 $a2 = 0.79$ ，即有0.21 mol H₂O发生了分解

写出热平衡

$$0 = [(1 - 0.1615 - 0.21)(\Delta_f H(H_2O) + \Delta H(H_2O))] + [b1(\Delta_f H(H) + \Delta H(H))] + [c1(\Delta_f H(OH) + \Delta H(OH))] + [b2\Delta H(H_2)] + [c2\Delta H(O_2)]$$

求得 $0 < 32831.1045$

插值得 $T = 3107$ K

作为验证我查询了deepseek提供的NASA CEA的数据，结果为3093K，仍存在细微的误差，应该来自于其他副反应，但主要的两个分解反应都已经考虑到，作为估算已十分准确，充分证明了JANAF表的可行，希望我的论文顺利见刊。
