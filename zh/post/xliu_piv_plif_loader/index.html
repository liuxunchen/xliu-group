PIV/PLIF Liutex 分析工具 – 基于 PyQt6 的流场涡识别与频谱分析软件

刘训臣

六月 29, 2026

引言

在实验流体力学中，粒子图像测速（PIV） 与 平面激光诱导荧光（PLIF） 是两种广泛使用的光学测量技术。前者能获得瞬时速度场，后者可捕捉标量（如浓度、温度）的空间分布。如何从大量实验数据中高效提取有物理意义的信息，尤其是分离出真实的涡结构，一直是个挑战。

近年来，Liutex 涡识别方法提供了一种严格、无阈值的旋转分解，能够从速度梯度张量中剥离出刚性旋转强度，避免传统判据（如 Q、λ₂）将剪切误判为涡的缺点。

Liutex方法简介

我们基于此理论，开发了一款跨平台、模块化的 GUI 软件，集成 PIV/PLIF 数据加载、Liutex 计算、多物理量可视化、动画播放以及单点频谱分析等功能。

软件主要功能

1. 多格式数据加载

  • PIV：支持 DaVis 输出的标准 .txt 文件（包括 #DaVis 头格式）以及 TECPLOT 风格的 .dat 文件。自动识别格式，无需手动指定。
  • PLIF：读取 TIFF 图像序列，并提供边缘保持滤波、阈值掩膜、模糊增强等预处理选项，便于与速度场叠加显示。
  • Bin 平均：可自定义下采样倍数，降低数据量并过滤高频噪声。
  • 原点平移：支持坐标系平移，方便与其它测量系统对齐。

2. 流场物理量计算

基于速度场 (u, v) 自动计算：

  • 基础量：合速度、涡量、散度、速度梯度分量 $ |\nabla u|, |\nabla v| $
  • Liutex：严格遵循刘超群教授的公式，输出纯旋转强度 \(R\)、剪切贡献 \(S\) 以及局部旋转角速度 $\lambda_{ci}$

计算核心完全与界面分离，可单独调用，便于批处理或嵌入其它脚本。

3. 可视化与交互

  • 使用 Matplotlib 嵌入 PyQt6 界面，利用 轮廓填色（contourf） 显示标量场，并可叠加 速度矢量（quiver） 和 流线（streamline）。
  • PLIF 叠加：将处理后的 PLIF 标量场以半透明形式覆盖在速度场之上，直观显示标量与流动结构的耦合。
  • 实时动画：支持播放/暂停、拖动滑块浏览序列帧。
  • 交互工具栏：Matplotlib 内置导航工具栏，支持缩放、平移、保存图片等操作。
  • 可随时将当前显示的标量场数据导出为 .npy 文件，便于后续分析。

4. 单点 FFT 分析

在流场图上点击任意位置，软件自动提取该点所有时间步的物理量（如 Liutex、涡量）序列，并进行 FFT 变换，绘制单边幅值谱。这对于识别流动中的周期性特征（如尾涡脱落频率）非常实用。

软件架构

项目遵循 核心逻辑与 GUI 分离 的设计原则：

piv_plif_loader/
├── app.py # 程序入口
├── core/ # 核心算法包（不依赖 Qt）
│ ├── piv_loader.py # PIV 数据加载器
│ ├── plif_loader.py # PLIF 数据加载与增强
│ ├── liutex.py # Liutex 计算
│ ├── field_calculations.py # 其他物理量计算
│ └── fft_analyzer.py # FFT 分析
├── gui/ # 界面包
│ ├── canvas.py # Matplotlib 画布与工具栏
│ ├── controls.py # 控制面板
│ ├── dialogs.py # FFT 结果对话框
│ └── main_window.py # 主窗口（整合信号与槽）
└── requirements.txt

核心模块可独立测试、复用；GUI 模块通过信号/槽机制响应控件变化，更新图形。该架构保证了代码的可维护性与可扩展性。

关键技术点

  • Liutex 求解精度：在梯度计算中对速度场进行了 NaN/Inf 过滤，并采用中心差分；对坐标间距使用了高精度浮点，避免因 float32 舍入导致网格不等距。
  • 坐标兼容性：自动识别并转置 PIV 输出格式，确保与 Matplotlib 的 contourf、streamplot 期望一致；流线绘制前使用 linspace 强制生成绝对等距坐标，避免因浮点误差导致的 'x' values must be equally spaced 错误。
  • 内存优化：所有数组均以 float32 存储，数百帧数据仍可流畅操作。

运行环境与依赖

  • Python ≥ 3.9
  • PyQt6
  • Matplotlib
  • NumPy, Pandas, SciPy
  • OpenCV (用于 PLIF 增强)
  • 可选：imageio（制作 GIF 动画）

可通过 pip install -r requirements.txt 一键安装依赖。软件已在 Linux (Fedora) 下测试通过，Windows/macOS 理论上也可运行。

快速上手

  1. 下载压缩包 piv_plif_loader
  2. 运行 python app.py
  3. 点击 “加载 PIV 序列” 选择 DaVis 导出的 .txt 或 .dat 文件（支持多选）
  4. 可选：“加载 PLIF 序列” 选择 TIFF 图像
  5. 在下拉框中选择要显示的物理量（涡量、散度、Liutex 等）
  6. 勾选 quiver 或 streamline 叠加矢量或流线
  7. 使用工具栏缩放/平移图像，或将数据保存为 .npy
  8. 点击 “在流场上选点” 后在图像上单击，即可查看该点的 FFT 频谱

本工具旨在为实验流体力学研究者提供一个轻量、直观且理论严谨的流场后处理平台。它特别适合处理高速 PIV/PLIF 时序数据，能有效分离出真实涡结构并分析其频谱特性。欢迎大家提出改进建议或贡献代码。
