反应流体光谱仿真程序 flame spectrum

刘训臣

设计思路

传统的光谱仿真需要用户手动查找数据库文件、输入各分子的浓度，这个过程繁琐且容易出错。我们的火焰–光谱联动工具将 Cantera 化学平衡计算 与 多分子 HITRAN 光谱仿真 无缝衔接，实现了“一键式”从燃烧条件到光谱预测的自动化流程。该程序作为 apps/flame_spectrum_apps.py 独立入口，内部调用已有的 HitranGUI 窗口完成光谱显示，完全复用了HITRAN 数据库用法之前开发的模块。

自动分子识别与浓度注入

联动工具最核心的改进在于无需手动选择 par 文件。它利用 HitranSpectrum.ISO 字典中内置的 55 种分子信息，将 Cantera 输出的物种名称（如 CO, H2O）自动映射到对应的分子 ID，并根据目录命名规则（01_H2O/, 05_CO/ 等）定位到相应的 .par 谱线文件。同时，勾选的物种摩尔分数直接从火焰结果中读取，自动填入光谱仿真界面，浓度单位转换为 ppm 显示。整个过程对用户完全透明。

实现的关键步骤：

  1. 在 FlameSpectrumApp 初始化时，从 HitranSpectrum.ISO 构建 名称 → 分子ID 的映射。
  2. 当用户点击“Launch HITRAN Spectrum Tool”时，遍历勾选的物种，根据分子 ID 和名称拼接文件夹路径，选取其中第一个 .par 文件。
  3. 将文件路径和摩尔分数组装成字典，调用 HitranGUI 新增的 auto_load_molecules() 方法批量加载。
  4. auto_load_molecules() 在后台填充文件表格和浓度控件，并自动更新波数范围。

使用流程

化学平衡仿真部分，气体的组分浓度/温度/压力，由化学平衡计算，用Cantera 开源程序计算。输入初始条件，例如温度压力组分，选择化学平衡的约束条件。得到平衡后的热力学状态，比如火焰温度。

  1. 设定燃烧条件
    打开 flame_spectrum_apps.py，在左侧面板选择燃料（如 CH₄）、氧化剂（空气、纯氧或自定义比例），设置当量比、初始温度和压力。

  2. 计算化学平衡
    点击 Calculate Equilibrium，Cantera 执行恒压绝热燃烧计算，右侧表格列出所有满足阈值（可调）的产物物种及其摩尔分数、ppm 值。

  3. 筛选目标物种
    默认自动勾选摩尔分数大于 1e-4 的物种，用户也可根据需求手动勾选需要仿真的分子（如 H₂O、CO₂、CO、NO 等）。

热力学状态对比 完成化学平衡计算后，界面右侧不仅列出平衡组分，还会在一个新增的表格中并排展示反应前（初始）与反应后（平衡）的热力学状态，包括压力、温度、摩尔体积、内能、焓和熵。所有能量单位均以 kJ/mol 或 kJ/(mol·K) 显示，便于直观分析燃烧过程的能量转化与温升。该对比表格与组分列表上下排列，一目了然。

  1. 一键启动光谱仿真
    点击 Launch HITRAN Spectrum Tool，程序自动完成以下操作：

    • 打开全新的 HitranGUI 窗口；
    • 将火焰温度、压力写入相应控件；
    • 根据勾选物种加载对应的 .par 文件，并填入摩尔分数转换后的 ppm 浓度；
    • 在日志区显示已加载的物种信息。

  2. 微调与计算
    在 HitranGUI 中，用户可以检查波数范围、分辨率、光程等参数，或手动增删其他分子，然后点击 开始计算混合光谱 即可获得该火焰条件下的高分辨率吸收/透射光谱。

数据库和计算模型

我的计算模型完全按照 HITRAN数据库用法 中介绍的公式。

现在大部分数据直接用HITRAN数据库计算，未来我们也会上传新的HITRAN格式的数据库。

扩展性

  • 若要仿真其它机理或燃料，只需在 GasCompositionSimulator 中加载对应的 .yaml 机理文件。
  • 对于数据库中未包含的分子（如 OH、O、H 等自由基），用户可以自行下载或生成 HITRAN 格式的 .par 文件放入相应文件夹，联动工具将自动识别（需扩充 ISO 字典）。
  • 未来计划支持 HITEMP（高温光谱数据库）、JPL 等数据库，进一步覆盖紫外至远红外波段。

下载与运行

软件包提供了完整的 Python 源代码和示例数据（含多个分子的 HITRAN 谱线）。运行环境需 Python ≥3.9 及 numpy, scipy, matplotlib, PyQt6, cantera。

# 克隆或下载项目后，进入目录
cd flame_spectrum

# 仅使用光谱仿真工具
python apps/hitran_gui.py

# 使用火焰‑光谱联动工具
python apps/flame_spectrum_apps.py

后记与展望

这个软件框架很灵活，只需要选择不同的数据库就可以模拟不同波段的光谱。

将原本单一的光谱计算脚本逐步演化为集燃烧化学平衡、多分子光谱合成、自动数据加载与专业可视化于一体的模块化工具。

未来将重点发展：

  • 高温光谱数据库：集成 HITEMP 或 ExoMol 数据库，准确模拟燃烧产物在高温下的谱线。
  • 自由基仿真：通过 JPL 或 CDMS 数据库生成紫外/可见波段的自由基光谱。
  • 实验对比：加入实验数据导入功能，方便定量比较仿真与测量结果。
