利用JANAF表估算氢氧火焰温度

高寅斌

八月 15, 2025

最近在着手准备有关JANAF表使用的相关论文，发现了一个有趣的问题，氢氧焰的燃烧温度应该怎么计算？这似乎并不难回答，按照热力学第一定律，反应前后的总焓保证不变即可，似乎很容易写出下面的热平衡：

$$\Delta_f H(H_2) + \frac{1}{2}\Delta_f H(O_2) = \Delta_f H(H_2O) + \Delta H(H_2O)$$

因为方程左边均为单质，这样看来水生成焓在数值上应该等于负的反应焓，查询JANAF表可以得知$\Delta_f H(H_2O)$等于-241.826kJ·mol⁻¹，我们可以查询多少温度下的焓变等于这个值的负值，结果是约5000K，按此结果我们做一下验算，我们看一下5000K下该反应的$\Delta G°$，$\Delta G°$取最小值的温度，就是标准状态下该反应平衡常数最大、反应进行得最彻底的理想温度，因此如果结果是5000K，$\Delta G°$应该为最小，带入数据，5000K时$\Delta_r G°(5000K) = \Delta_f G°(H₂O, g, 5000K) = 40.949$ kJ·mol⁻¹，4900K时$\Delta_r G°(4900K) = \Delta_f G°(H₂O, g, 4900K) = 34.998$kJ·mol⁻¹，很显然5000K并不是理想温度，且高出非常多，那么这个答案必然存在问题，我们不妨仔细分析一下这个问题：

首先得明确核心约束：定HP意味着产物的总焓必须和反应物初始总焓完全匹配，而温度和组分都是这个约束下的结果，不是提前固定的。所以要先做一件基础工作——计算反应物的总焓：根据氢氧的混合比例（比如化学计量比2:1）、初始温度（比如室温298K）和设定的压力，算出反应物的总焓值，这个值是后续所有筛选的“基准线”。接下来就是先划定一个合理的温度范围，然后按一定步长逐个遍历这个区间里的每个温度。对每个遍历到的温度，都要在“固定这个温度+固定设定压力”的前提下，让产物的组分自由调整（但必须满足氢、氧原子总数不变的元素守恒），直到找到吉布斯自由能最小的状态——这个状态就是该温度下的化学平衡组分，意味着此时的组分比例是最稳定的，再没有自发变化的趋势。之后，需要计算这个“特定温度+对应平衡组分”组合下的产物总焓$H_{prod}$，把它和之前算的反应物总焓基准线做对比。等把所有温度都遍历完，就会得到一系列“温度-产物焓”的对应关系，而要找的那个真实燃烧温度，就是其中产物焓和反应物总焓最接近（甚至相等）的那个温度——如果刚好有某个扫描温度对应的产物焓和基准线完全匹配，那这个温度就是定HP下的平衡温度，对应的组分就是最终答案；如果没有完全匹配的用相邻两个温度的焓值做线性插值，就能算出精确的平衡温度，以及对应的平衡组分。

反应物配比为化学计量比（2 mol H₂ + 1 mol O₂），初始温度为 \( T_r = 298.15\ \text{K} \)，单物质摩尔焓满足：

$$ H_i(T_r) = \Delta H_i(T_r) + \Delta_f H_i(T_r) $$

（H₂、O₂为单质，$\Delta_f H_i(T_r) = 0$）

$$ H_{\text{react}} = 2 \cdot H_{\text{H}_2}(T_r) + 1 \cdot H_{\text{O}_2}(T_r) $$$$2n_{\text{H}_2\text{O}} + 2n_{\text{H}_2} = 4 \implies n_{\text{H}_2} = 2 - n_{\text{H}_2\text{O}}$$$$n_{\text{H}_2\text{O}} + 2n_{\text{O}_2} = 2 \implies n_{\text{O}_2} = \frac{2 - n_{\text{H}_2\text{O}}}{2}$$$$n_{\text{total}} = n_{\text{H}_2\text{O}} + n_{\text{H}_2} + n_{\text{O}_2}$$

总吉布斯自由能表达式：

$$\begin{align*} G_{\text{total}}(T,P,n_{\text{H}_2\text{O}}) &= n_{\text{H}_2\text{O}} \left[ \Delta_f G_{\text{H}_2\text{O}}(T) + R T \ln\left(\frac{n_{\text{H}_2\text{O}}}{n_{\text{total}}} \cdot \frac{P}{P_0}\right) \right] \\ &+ (2 - n_{\text{H}_2\text{O}}) \left[ \Delta_f G_{\text{H}_2}(T) + R T \ln\left(\frac{2 - n_{\text{H}_2\text{O}}}{n_{\text{total}}} \cdot \frac{P}{P_0}\right) \right] \\ &+ \frac{2 - n_{\text{H}_2\text{O}}}{2} \left[ \Delta_f G_{\text{O}_2}(T) + R T \ln\left(\frac{2 - n_{\text{H}_2\text{O}}}{2n_{\text{total}}} \cdot \frac{P}{P_0}\right) \right] \end{align*}$$

平衡组分优化目标：

$$\min_{n_{\text{H}_2\text{O}}} G_{\text{total}}(T,P,n_{\text{H}_2\text{O}}) \quad \text{s.t.} \quad 0 \leq n_{\text{H}_2\text{O}} \leq 2$$

解得平衡组分：

$$n_{\text{H}_2\text{O}}^*、n_{\text{H}_2}^* = 2 - n_{\text{H}_2\text{O}}^*、n_{\text{O}_2}^* = \frac{2 - n_{\text{H}_2\text{O}}^*}{2}$$

产物总焓计算： 物质 $i$ 在温度 $T$ 下的摩尔焓：

$$H_i(T) = \Delta H_i(T) + \Delta_f H_i(T)$$

平衡组分对应的总焓：

$$H_{\text{product}}(T) = n_{\text{H}_2\text{O}}^* \cdot H_{\text{H}_2\text{O}}(T) + n_{\text{H}_2}^* \cdot H_{\text{H}_2}(T) + n_{\text{O}_2}^* \cdot H_{\text{O}_2}(T)$$

定HP平衡温度插值： 找到相邻温度 $T_a < T_b$ 满足 $H_{\text{product}}(T_a) \leq H_{\text{react}} \leq H_{\text{product}}(T_b)$，线性插值求解 $T_{\text{eq}}$： $T_{\text{eq}} = T_a + \frac{H_{\text{react}} - H_{\text{product}}(T_a)}{H_{\text{product}}(T_b) - H_{\text{product}}(T_a)} \cdot (T_b - T_a)$ 我们可以使用python代码完成这些流程，得到的结果部分如下： 正在计算 4700K 平衡组分… 产物总焓：355.18 kJ | 平衡组分（H2O,H2,O2）：[0.4524 1.5476 0.7738]

这就是扫描的方法，但这样的方法无疑耗时又费力，那么可以在此基础上优化一下，采用“温度和组分一起迭代求解”的策略：

首先，给出一个温度的初始猜测值$T_{guess}$（例如3000 K）。假设不同的组分可以求解在该$T_{guess}$下的平衡组成$\xi_{eq}$。用得到的$(T_{guess}, \xi_{eq})$计算产物总焓$H_{prod}$。判断：若$H_{prod} < 0$，说明产物焓太低，系统能量有盈余，应升高$T_{guess}$；若$H_{prod} > 0$，说明产物焓太高，能量不足，应降低$T_{guess}$。根据差值大小，更新$T_{guess}$，直到$|H_{prod}|$足够接近0。

内层循环：在固定温度$T$下，求平衡组成$\xi$。假设组分变化，比如氢气2mol，氧气1mol，然后氢气反应了$A$，剩余$2-A$mol，氧气反应了$1-1/2A$mol，生成了$A$mol的水，类似地扫描不同的组分，按照外层猜测的温度，从JANAF表查出该温度下$H_2, O_2, H_2O$的$\Delta_f G°$。计算此温度下的平衡常数$K = exp(-\Delta G°/(RT))$。求解化学平衡方程$K = [\xi / ((2-\xi) \times sqrt(1-0.5\xi))] \times sqrt(3-0.5\xi)$得到$\xi_{eq}$。这可以通过扫描平衡计算方程左右两边的差值，寻找使差值最小的$\xi$来实现，这等价于寻找系统总吉布斯自由能$G$最小的点。

确定了求解的基本方法之后，就可以引入更多的产物而不仅仅是水，因为高温下氢氧燃烧存在显著的解离反应，忽略这些解离产物会导致温度和组分的计算结果与实际严重偏离，最开始时求解温度约5000K很重要的一点原因就是这个问题。

下面来完整的算一下，所用到的数据均来自JANAF表：

由于$H_2O$在高温下会发生分解，则分别写出主要的分解方程式：

$$H_2 + \frac{1}{2}O_2 \to a1H_2O + b1H + c1OH$$$$\begin{cases}2 = 2a1 + b1 + c1 \\1 = a1 + c1 \end{cases} \Rightarrow \begin{cases}b1 = 1 - a1 \\c1 = 1 - a1 \end{cases}$$$$H_2O \to H + OH$$$$K = \frac{bc}{a} = \frac{(1 - a1)^2}{a1}$$$$H_2 + \frac{1}{2}O_2 \to a2H_2O + b2H_2 + c2O_2$$$$\begin{cases}2 = 2a2 + 2b2 \\1 = a2 + 2c2 \end{cases} \Rightarrow \begin{cases}b2 = 1 - a2 \\c2 = \frac{1 - a2}{2} \end{cases}$$$$H_2O \to H_2 + \frac{1}{2}O_2$$$$K = \frac{b2c2^{\frac{1}{2}}}{a2} = \frac{(1 - a2)(\frac{1 - a2}{2})^{\frac{1}{2}}}{a2}$$

第一步：假设产物温度为3100K

  • 对反应1：查询JANAF表得$\Delta_f H(H_2O) = -71298$（J·mol⁻¹），$\Delta_f H(H) = 39877$（J·mol⁻¹），$\Delta_f H(OH) = -5552$（J·mol⁻¹），计算得$\Delta G_1 = 105623$（J·mol⁻¹）。则平衡常数$K = e^{-\frac{\Delta G}{RT}} = 0.0166$，由$K = \frac{(1 - a1)^2}{a1}$求得$a1 = 0.879$，即有0.121 mol的水发生了分解，产物为H和OH。

  • 对反应2：查询JANAF表得$\Delta_f H(H_2O) = -71298$（J·mol⁻¹），$\Delta_f H(H_2) = \Delta_f H(O_2) = 0$（J·mol⁻¹），计算得$\Delta G_2 = 71298$（J·mol⁻¹）。则平衡常数$K = e^{-\frac{\Delta G}{RT}} = 0.0629$，由$K = \frac{(1 - a2)(\frac{1 - a2}{2})^{\frac{1}{2}}}{a2}$求得$a2 = 0.825$，即有0.175 mol的水分解，产物为$H_2$和$O_2$。

由热力学第一定律，反应前后总焓不变，写出热平衡方程：

$$0 = [(1 - 0.121 - 0.175)(\Delta_f H(H_2O) + \Delta H(H_2O))] + [b1(\Delta_f H(H) + \Delta H(H))] + [c1(\Delta_f H(OH) + \Delta H(OH))] + [b2\Delta H(H_2)] + [c2\Delta H(O_2)]$$

计算结果：$0 > -2688.0705$（J·mol⁻¹），说明产物焓偏低，需升高温度。

第二步：假设产物温度为3200K

  • 对反应1：查询JANAF表得$\Delta G_1 = 92313$（J·mol⁻¹），则平衡常数$K = e^{-\frac{\Delta G}{RT}} = 0.0311$，由$K = \frac{(1 - a1)^2}{a1}$求得$a1 = 0.8385$，即有0.1615 mol $H_2O$发生了分解。

  • 对反应2：查询JANAF表得$\Delta G_2 = 65430$（J·mol⁻¹），则平衡常数$K = e^{-\frac{\Delta G}{RT}} = 0.0855$，由$K = \frac{(1 - a2)(\frac{1 - a2}{2})^{\frac{1}{2}}}{a2}$求得$a2 = 0.79$，即有0.21 mol $H_2O$发生了分解。

由热力学第一定律，写出热平衡方程：

$$0 = [(1 - 0.1615 - 0.21)(\Delta_f H(H_2O) + \Delta H(H_2O))] + [b1(\Delta_f H(H) + \Delta H(H))] + [c1(\Delta_f H(OH) + \Delta H(OH))] + [b2\Delta H(H_2)] + [c2\Delta H(O_2)]$$

计算结果：$0 < 32831.1045$（J·mol⁻¹），说明产物焓偏高，需降低温度。

第三步：线性插值求平衡温度

根据3100K（焓差-2688.0705 J·mol⁻¹）和3200K（焓差32831.1045 J·mol⁻¹）的计算结果，线性插值得：

$$T = 3107 \text{ K}$$

作为验证，我使用Cantera在HP约束下计算，结果为3093K，仍存在细微的误差，应该来自于未考虑其他副反应（如$H_2O_2$、$HO_2$等次要组分的生成），但主要的两个分解反应都已纳入计算，作为估算已十分准确，充分证明了JANAF表在氢氧火焰温度估算中的可行性。希望我的论文顺利见刊。
