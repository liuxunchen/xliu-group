利用JANAF表估算氢氧火焰温度

高寅斌

八月 15, 2025

最近在着手准备有关JANAF表使用的相关论文，发现了一个有趣的问题，氢氧焰的燃烧温度应该怎么计算？这似乎并不难回答，按照热力学第一定律，反应前后的总焓保证不变即可，似乎很容易写出下面的热平衡：

$$\Delta_f H(H_2) + \frac{1}{2}\Delta_f H(O_2) = \Delta_f H(H_2O) + \Delta H(H_2O)$$

因为方程左边均为单质，这样看来水生成焓在数值上应该等于负的反应焓，查询JANAF表可以得知$\Delta_f H(H_2O)$等于-241.826kJ·mol⁻¹，我们可以查询多少温度下的焓变等于这个值的负值，结果是5200K，看起来这个问题就这么解决了，但很明显这个答案与现实不符，一当量比的氢氧焰不可能有这么高的温度，在与老师讨论后发现了问题所在，水在高温下的分解没有被考虑，那么考虑分解结果如何呢？下面来完整的算一下，所用到的数据均来自JANAF表：

由于$H_2O$在高温下会发生分解，则分别写出主要的分解方程式

$$H_2 + \frac{1}{2}O_2 \to a1H_2O + b1H + c1OH$$$$\begin{cases}2 = 2a1 + b1 + c1 \\1 = a1 + c1 \end{cases} \Rightarrow \begin{cases}b1 = 1 - a1 \\c1 = 1 - a1 \end{cases}$$$$H_2O \to H + OH$$$$K = \frac{bc}{a} = \frac{(1 - a1)^2}{a1}$$$$H_2 + \frac{1}{2}O_2 \to a2H_2O + b2H_2 + c2O_2$$$$\begin{cases}2 = 2a2 + 2b2 \\1 = a2 + 2c2 \end{cases} \Rightarrow \begin{cases}b2 = 1 - a2 \\c2 = \frac{1 - a2}{2} \end{cases}$$$$H_2O \to H_2 + \frac{1}{2}O_2$$$$K = \frac{b2c2^{\frac{1}{2}}}{a2} = \frac{(1 - a2)(\frac{1 - a2}{2})^{\frac{1}{2}}}{a2}$$

设产物温度为3100K

对反应1：查询JANAF表得：$\Delta_f H(H_2O) = -71298$ ，$\Delta_f H(H) = 39877$ ，$\Delta_f H(OH) = -5552$ ，则 $\Delta G_1 = 105623$

$$K = e^{-\frac{\Delta G}{RT}} = 0.0166$$

，由 $K = \frac{(1 - a1)^2}{a1}$ 求得 $a1 = 0.879$

即有0.121 mol的水发生了分解，产物为H和OH

对反应2：查询JANAF表得 $\Delta_f H(H_2O) = -71298$ ，$\Delta_f H(H_2) = \Delta_f H(O_2) = 0$ ，则 $\Delta G_2 = 71298$

$$K = e^{-\frac{\Delta G}{RT}} = 0.0629$$

，求得 $a2 = 0.825$

即有0.175 mol的水分解，产物为H₂和O₂

由热力学第1定律，反应前后总焓不变

写出热平衡

$$0 = [(1 - 0.121 - 0.175)(\Delta_f H(H_2O) + \Delta H(H_2O))] + [b1(\Delta_f H(H) + \Delta H(H))] + [c1(\Delta_f H(OH) + \Delta H(OH))] + [b2\Delta H(H_2)] + [c2\Delta H(O_2)]$$

求得 $0 > -2688.0705$

设产物温度3200K

$$K = e^{-\frac{\Delta G}{RT}} = 0.0311$$

由 $K = \frac{(1 - a1)^2}{a1}$ 求得 $a1 = 0.8385$ ，即有0.1615 mol H₂O发生了分解

对反应2：$\Delta G_2 = 65430$ ，则 $K = 0.0855$

由 $K = \frac{(1 - a2)(\frac{1 - a2}{2})^{\frac{1}{2}}}{a2}$ 求得 $a2 = 0.79$ ，即有0.21 mol H₂O发生了分解

写出热平衡

$$0 = [(1 - 0.1615 - 0.21)(\Delta_f H(H_2O) + \Delta H(H_2O))] + [b1(\Delta_f H(H) + \Delta H(H))] + [c1(\Delta_f H(OH) + \Delta H(OH))] + [b2\Delta H(H_2)] + [c2\Delta H(O_2)]$$

求得 $0 < 32831.1045$

插值得 $T = 3107$ K

作为验证我查询了deepseek提供的NASA CEA的数据，结果为3093K，仍存在细微的误差，应该来自于其他副反应，但主要的两个分解反应都已经考虑到，作为估算已十分准确，充分证明了JANAF表的可行，希望我的论文顺利见刊。
