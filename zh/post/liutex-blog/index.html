liutex在二维流场的应用

王硕

十一月 11, 2025

近期阅读了《Liutex-涡定义和第三代涡识别方法》这篇文章，liutex方法是为三维流场设计的，它的求解包含三维速度梯度张量分解、RS 分解等步骤，理论的核心逻辑可概括为三点：1.从速度梯度张量中提取旋转相关的共轭复特征值；2.通过数学运算剥离剪切、拉伸等非旋转成分；3.最终得到仅反映刚性旋转的 Liutex 向量（大小 + 方向），根据这个核心逻辑，可以针对性简化，用于求解二维流场的liutex，处理二维流场的数据。 文章中强调，速度梯度张量是涡识别的基础，其特征值包含了流场旋转、拉伸、剪切的全部信息。 二维流场中，通过np.gradient计算速度梯度张量的 4 个偏导数，

    dUdY = np.gradient(U, Y[:, 0], axis=0, edge_order=1)
    dVdX = np.gradient(V, X[0, :], axis=1, edge_order=1)
    dUdX = np.gradient(U, X[0, :], axis=1, edge_order=1)
    dVdY = np.gradient(V, Y[:, 0], axis=0, edge_order=1)
    grad_tensor = np.zeros((U.shape[0], U.shape[1], 2, 2), dtype=np.float32)
    grad_tensor[:, :, 0, 0] = dUdX
    grad_tensor[:, :, 0, 1] = dUdY
    grad_tensor[:, :, 1, 0] = dVdX
    grad_tensor[:, :, 1, 1] = dVdY

组装为 2×2 矩阵：

$$ \nabla \vec{v} = \left[ \begin{array}{cc} % cc表示2列，均居中对齐 \frac{\partial U}{\partial X} & \frac{\partial U}{\partial Y} \\ \frac{\partial V}{\partial X} & \frac{\partial V}{\partial Y} \end{array} \right] $$

当流场存在刚性旋转时，速度梯度张量会存在共轭复特征值，其虚部直接影响旋转的强度。在二维流场中，通过特征值分解并提取虚部：

   eig_vals = np.linalg.eigvals(grad_tensor)
    wi0 = eig_vals[:, :, 0].imag
    wi1 = eig_vals[:, :, 1].imag

由于共轭复特征值的虚部大小相等、符号相反，wi0 * wi1的绝对值恰好是虚部大小的平方，这一步为后续剥离剪切成分提供了关键依据。 三维流场中，Liutex 是向量，其方向为速度梯度张量的实特征向量。二维流场中旋转轴固定，用涡量的符号来表征旋转方向（当vor＞0是逆时针旋转，值为正，相反vor＜0是顺时针旋转，值为负）

    vor = dVdX - dUdY
    vor_sign = np.where(vor > 0, 1.0, np.where(vor < 0, -1.0, 0.0))

vor=R+S，即vor是旋转与剪切/拉伸的总和，sqrt_term是剪切/拉伸量，liutex的大小通过vor与sqrt_term的差值来求得。

   sqrt_term = np.sqrt(np.abs(np.square(vor) - 4 * np.abs(wi0 * wi1)))
    R = (np.abs(vor) - sqrt_term) * vor_sign
    stretch = sqrt_term
