喷雾场 PIV 分析：从图像到速度场全流程

在燃烧与喷雾诊断研究中，粒子图像测速（PIV）是获取流场速度分布的核心手段。本文将结合实际喷雾图像，完整复现从原始图像预处理、PIV 计算到结果可视化的全流程，帮助快速上手 OpenPIV 工具。

1. 准备工作：图像读取与预处理1. 准备工作：图像读取与预处理

我们的实验对象是两帧连续的喷雾图像，记录了燃烧室中液滴的运动。首先需要安全读取图像并进行预处理，以提升后续 PIV 计算的精度。 1.1 图像读取 为了避免 TIFF 图像的 LZW 压缩问题，我们使用 PIL 库来读取图像，并将其转换为灰度图

可以看到，图像中包含了清晰的喷雾粒子和燃烧室背景，但整体对比度偏低，不利于后续的互相关计算。

1.2 图像预处理 我们对图像进行对比度拉伸，增强粒子与背景的区分度 处理后的图像对比如下：

经过处理，喷雾粒子的边缘更加锐利，灰度分布也更加均匀，为后续的 PIV 计算打下了良好基础。

2. 核心计算：PIV 速度场求解2. 核心计算：PIV 速度场求解

OpenPIV 的核心是基于互相关算法的速度场计算。我们设置了关键参数： 窗口大小 (winsize): 32 像素 搜索区域 (searchsize): 32 像素 重叠率 (overlap): 16 像素 (50%) 时间间隔 (dt): 0.02 秒 2.1 数据验证与后处理 原始计算结果中可能存在异常值，我们通过信噪比（sig2noise）进行验证，并使用局部平均法替换异常值。

3. 结果可视化：从矢量场到统计分析3. 结果可视化：从矢量场到统计分析

计算完成后，我们需要通过丰富的可视化手段来解读流场信息。 3.1 速度矢量场 最直观的方式是直接绘制速度矢量场，我们可以将其叠加在原始图像上，清晰地看到喷雾粒子的运动方向和强度。 生成的三合一矢量场分析图如下：

从图中可以清晰地看到： 喷雾场在燃烧室出口处（图像右侧）速度明显增大，形成了高速射流。 中心区域（喷孔附近）的速度分布较为复杂，存在明显的涡结构。 3.2 详细统计分析 为了更深入地分析流场特性，我们生成了包含多种统计图表的综合分析图： 这张图包含了丰富的信息： 速度分量分布：U、V 分量和速度大小的概率密度分布，显示了流场的整体统计特性。 空间分布：速度大小的空间分布散点图，直观展示了高速区域的位置。 方向分布：速度方向的直方图，揭示了流场的主导流动方向。 流线图：通过插值生成的流线图，清晰地描绘了流场的涡结构和流动模式。 速度剖面：在 Y=3 和 X=7 处的速度剖面，用于定量分析特定位置的流场特性。

4. 结果保存与报告生成4. 结果保存与报告生成

最后，我们将 PIV 结果保存为文本和 CSV 格式，方便后续分析，并自动生成一份包含所有关键信息的分析报告。 总结 通过 OpenPIV，我们成功地从两帧喷雾图像中提取出了完整的速度场信息，并通过多维度的可视化手段对其进行了深入分析。这个流程不仅适用于喷雾场，也可以推广到其他流场的 PIV 分析中。
